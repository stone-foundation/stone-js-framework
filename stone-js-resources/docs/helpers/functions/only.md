# Function: only()

> **only**\<`T`\>(`object`, `keys`): `Partial`\<`T`\>

### T

`T` *extends* [`ResourceOutput`](../../declarations/type-aliases/ResourceOutput.md)

## Parameters

### object

`T`

The source object.

### keys

`string`[]

The keys to keep.

## Returns

`Partial`\<`T`\>
