# Changelog

This monorepo releases all `@stone-js/*` packages together (lockstep). Versions and per-package
changelogs are generated by [Changesets](https://github.com/changesets/changesets).

- Per-package changelogs live in each package folder: `stone-js-<name>/CHANGELOG.md`.
- To record a change: `pnpm changeset` (see [MONOREPO.md](./MONOREPO.md)).

## 0.8.0 — unreleased

First lockstep release of the consolidated monorepo. Highlights across the framework:

- **Kernel**: platform-agnostic hardening, deep blueprint merge fixes, identity-based provider dedupe.
- **Router**: ReDoS elimination (linear path patterns), named-group binding, URL-encoding in
  `generate()`, precompiled regexes, deterministic specificity ordering, `Allow` header on 405.
- **http-core**: cookie-secret hardening, default upload limits, header-count / DoS guards.
- **Adapters**: safe method-override, `rawBody` in metadata, idempotent process handlers.
- **Frontend**: agnostic view layer (`use-view`), standardized React components & hooks, view-layer
  error rendering, View Transitions.
- **Tooling**: extensible starter-plugin contract, abstract Logger layer, `@stone-js/telemetry`.

See each package's `CHANGELOG.md` for details.
